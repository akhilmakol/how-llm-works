# LinkedIn Launch Post

## Personal Post Variant

I'm excited to share **how-llm-works**: a visual, interactive open source project that explains how Large Language Models work from first principles.

I built it to make LLM internals easier to understand without oversimplifying the architecture.

The repository includes:

- a mini GPT-style model in PyTorch
- visual diagrams for tokenization, embeddings, attention, transformers, training, and inference
- beginner-friendly docs and notebooks
- a Streamlit app for text generation, token exploration, and attention visualization
- a banking-themed learning flow to make examples more concrete and grounded

The goal was to create something useful for:

- learners exploring LLMs for the first time
- educators teaching transformer fundamentals
- builders who want a compact, readable reference implementation

If you've ever wanted to understand what happens between raw text, tokens, attention, and next-token prediction, this project walks through the whole pipeline end to end.

Repo:
https://github.com/akhilmakol/how-llm-works/tree/master

I'd love feedback from engineers, researchers, educators, and anyone building practical AI learning tools.
