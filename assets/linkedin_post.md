# LinkedIn Post Draft

I built `how-llm-works`, a visual-first educational project that explains large language models from first principles.

What it includes:

- a mini GPT-style model in PyTorch
- beginner-friendly docs for tokens, embeddings, attention, training, and inference
- a Streamlit app for text generation and attention visualization
- simple tests and notebooks for exploration

The goal was to make LLM internals feel approachable without hiding the real architecture.
