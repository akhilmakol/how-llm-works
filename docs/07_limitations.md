# 7. Limitations

This mini project is educational, not state of the art.

## Important Limits

- the dataset is tiny
- the tokenizer is only word-level
- the model is very small
- greedy decoding can be repetitive
- training on one file does not create broad knowledge

## What This Means

The model may:

- repeat phrases
- produce short or narrow outputs
- fail on unfamiliar prompts
- memorize patterns from the sample text

## Why This Is Still Useful

Even with these limits, the project teaches the real building blocks of larger systems.

Understanding the small version makes the large version much easier to reason about.
